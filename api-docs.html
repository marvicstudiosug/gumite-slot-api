📡 Endpoints

GET /api/health
Check if the API is running
Response: { "status": "online", "version": "1.0.0", "timestamp": 123456789 }
GET /api/info
Get game information (symbols, payouts, house edge)
Response: { "name": "GUMITE Slot", "houseEdge": 0.065, "symbols": [...] }
POST /api/spin
Perform a spin (requires API key)
ParameterTypeRequiredDescription
playerIdstringrequiredUnique player identifier
betAmountnumberrequiredBet amount in UGX (10-10,000)
Request: { "playerId": "player_123", "betAmount": 500 }
Response: { "success": true, "spinId": "spin_123", "grid": ["🍒","🍋","🍊",...], "win": 0, ... }
POST /api/deposit
Deposit funds into your company account
Request: { "amount": 10000, "reference": "DEP_123" }
Response: { "success": true, "transactionId": "DEP_123", "newBalance": 15000 }
POST /api/withdraw
Withdraw funds from your company account
Request: { "amount": 5000, "bankAccount": "ACC123456", "reference": "WITHDRAW_123" }
Response: { "success": true, "transactionId": "WITHDRAW_123", "newBalance": 10000 }
GET /api/balance
Check your company balance
Response: { "companyId": "comp_123", "companyName": "Test Casino", "balance": 15000, "currency": "UGX" }
GET /api/stats
Get game statistics for your company
Response: { "companyId": "comp_123", "totalSpins": 1234, "winRate": 28.5, "profit": 4567, ... }
